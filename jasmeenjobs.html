Cybersecurity analyst


Cybersecurity analysts assess, plan and implement security measures to help protect an organisation
from security breaches and attacks on its computer networks and systems.
This role includes simulating vulnerability attacks, reviewing new technologies to protect company data,
and encouraging users to interact with the latest regulations and procedures to ensure
that the network remains secure.

Cyber-security researchers are typically working for various companies as part of an IT department and generally work in full-time.
This work is most suited for people with outstanding programming skills and expertise and
who want to think creatively and find new solutions.




Responsibilities include:
  • Keep updated with the latest developments in security and technology.
  • Research/evaluation of emerging cybersecurity threats and ways of handling them.
  • Prepare for the recovery of disasters and develop contingency plans for any security breaches.
  • Monitoring of attacks, intrusions and rare, irregular or unwanted activity.
  • Test and evaluate security products.
  • Design new security systems or upgrade existing ones
  • Use advanced analytic tools to determine emerging threat patterns and vulnerabilities
  • Identify potential weaknesses and implement measures, such as firewalls and encryption
  • Monitor and respond to 'phishing' emails and 'pharming' activity
  • Assist with the development, maintenance and delivery of training for colleagues on cybersecurity awareness
  • Offer employees advice and feedback on matters such as spam and unwanted or harmful emails.


Qualification

  • Hold at least a bachelor’s degree in computer science, engineering, mathematics, or another computer-related field.
  • Some employers prefer applicants who have a Master of Business Administration (MBA) in information systems. Programs offering the MBA in information systems generally require 2 years of study beyond the undergraduate level and include both business and computer-related courses.



Software engineer



Software engineering is a branch of computer science which includes the development and building of computer systems software and applications software.

Computer systems software is made up of programmes that include computer utilities and operating systems. Applications software consists of user-focused programmes including web browsers, database programmes, etc.


Software Engineer Responsibilities

  • Improving system quality by identifying issues and common patterns, and developing standard operating procedures.
  • Enhancing applications by identifying opportunities for improvement, making recommendations and designing and implementing systems.
  • Developing and directing validation and testing methods for the software system.
  • Directing initiatives for software programming
  • Oversee the development of the documentation.
  • Write well designed, testable codes.
  • Document and maintain the performance of the software
  • Troubleshooting, debugging and updating existing systems
  • Install programmes and evaluate feedback from users
  • Comply with project plans and industry standards;
  • Ensure that the software is updated with the latest features
  • Design and develop a processing platform using various configuration management technologies.
  • Collaborate cross-functionally with data scientists, business users, project managers and other engineers to achieve elegant solutions.


Qualification and Skills Needed
  • Analyzing information
  • General programming skills
  • Software design
  • Software debugging
  • Software documentation
  • Software testing
  • Problem-solving skills
  • Teamwork
  • Software development fundamentals
  • Software development process
  • Software requirements
Education and experience Needed
  • Bachelor’s and/or Master’s degree in Computer Science, Computer Engineering or related technical discipline 5+ years of professional software development experience
  • Experience using relevant tool suites
